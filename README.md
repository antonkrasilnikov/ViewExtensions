# ViewExtentions

Collection of UIKit extensions and helpers

 [TableView](./Source/ViewExtensions/TableView/) a declarative and extensible wrapper around `UITableView` that simplifies building complex table-based UIs.


 [CollectionView](./Source/ViewExtensions/CollectionView/) a declarative and extensible wrapper around `UICollectionView` that simplifies building complex collection-based UIs.


 [Animation](./Source/ViewExtensions/Animations/) a lightweight utility library that simplifies working with Core Animation on iOS.
